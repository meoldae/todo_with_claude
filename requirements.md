# 요구사항 분석

## 1. 프로젝트 개요

할 일 관리와 가계부 기능을 통합한 개인 생산성 앱.  
일상의 할 일 관리와 지출·수입 기록을 하나의 서비스에서 처리하고, 연말정산 준비까지 지원하는 것을 목표로 한다.

---

## 2. 개발 단계

| 단계 | 내용 |
|------|------|
| 1단계 | 투두 앱 (PC Windows 앱) |
| 2단계 | 가계부 기능 통합 (PC Windows 앱) |
| 3단계 | 모바일 앱 확장 (Android / iOS) |

---

## 3. 1단계 — 투두 앱

### 3.1 핵심 기능

- **할 일 생성·수정·삭제**
  - 제목, 메모, 마감일, 우선순위 설정
- **완료 처리**
  - 각 항목 맨 앞에 체크박스 배치 — 클릭/터치 한 번으로 완료 상태 토글 (Microsoft Todo 방식)
  - 완료된 항목은 시각적으로 구분 (취소선, 흐림 처리 등)
  - 완료 항목은 목록 하단으로 분리하거나 접어서 표시
- **실행 취소(Undo)**
  - 완료 처리 또는 삭제 직후 토스트 메시지와 함께 Undo 버튼 노출
  - 일정 시간(수 초) 내에 클릭하면 이전 상태로 복구
- **목록 관리**
  - 카테고리 또는 프로젝트 단위로 할 일 묶기
  - 드래그 앤 드롭으로 순서 변경
- **필터 및 검색**
  - 날짜별, 카테고리별, 우선순위별 필터
  - 키워드 검색
- **반복 일정**
  - 매일 / 매주 / 매월 반복 설정

### 3.2 부가 기능

- 오늘 할 일 대시보드 (Today 뷰)
- 마감 임박 항목 알림
- 완료율 통계 (주간·월간)

---

## 4. 2단계 — 가계부 기능

### 4.1 기본 지출·수입 관리

- 수입·지출 내역 기록 (금액, 날짜, 카테고리, 메모)
- 거래 내역 수정·삭제
- 카테고리 직접 관리 (식비, 교통, 의료, 문화 등)
- 월별·연별 수지 요약

### 4.2 예산 관리

- 카테고리별 월간 예산 설정
- 예산 대비 지출 현황 시각화
- 예산 초과 시 경고

### 4.3 통계 및 분석

- 월별·카테고리별 지출 추이 그래프
- 전월 대비 증감 비교
- 고정 지출 vs 변동 지출 구분

### 4.4 연말정산 지원

- **공제 항목 자동 분류**
  - 의료비, 교육비, 기부금, 신용카드·체크카드 사용액 등 연말정산 공제 항목에 해당하는 지출 자동 태깅
- **연간 공제 현황 대시보드**
  - 항목별 누적 지출액과 공제 한도 대비 현황 표시
  - 남은 한도 안내
- **연말정산 보고서 내보내기**
  - 공제 항목별 지출 내역을 연도별로 정리해 PDF 또는 CSV로 내보내기
- **체크카드 vs 신용카드 사용 비율 추적**
  - 소득공제율 차이를 고려한 사용 비율 모니터링 및 조언
- **의료비 공제 한도 추적**
  - 총급여 대비 의료비 공제 가능 여부 안내

---

## 5. 3단계 — 모바일 앱 확장

- 1·2단계에서 구현된 모든 기능을 Android·iOS에서 동일하게 사용 가능
- 모바일 특화 기능
  - 푸시 알림 (마감일, 예산 초과 등)
  - 영수증 사진 촬영 후 지출 내역 자동 입력 (OCR)
  - 위젯 지원 (오늘 할 일, 이번 달 지출 요약)

---

## 6. 공통 요구사항

- **1·2단계**: 데이터는 로컬 기기에 저장 (서버 불필요)
- **3단계**: 계정 연동을 통해 PC(Windows)·모바일 간 동기화 (BaaS 도입 검토)
- 엔티티 구조와 API 규격은 1단계부터 서버 전환을 고려해 통일된 형태로 설계
- 다크 모드 지원
- 데이터 백업·복원 기능
